# tof_detection_module
AFP Project 5


## Prerequisite
- Python 3.10
- ROS 2 Humble


## Installation

1. Setup ROS ws
```shell
mkdir -p ~/ros2_ws/src
cd ~/ros2_ws/src
```

2. Clone repository
```shell
git clone git@github.com:Ferdinand50/tof_detection_module.git
```

3. Build ws
```shell
colcon build
```
